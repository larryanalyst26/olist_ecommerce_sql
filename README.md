# olist_ecommerce_sql
I use SQL with DB Browser for SQLite to analyze a dataset from Olist on Brazilian ecommerce. The dataset is too large to upload to Github, but it is free to download from Kaggle. Here is the link: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

This dataset is from the company Olist directly and on Kaggle. There are many different queries that one could do with the dataset, but I focused on queries that would be valuable for the company Olist. Olist is a company that provides ecommerce assistance to ecommerce sellers. They offer logistics and platform help. They are not a marketplace themselves. They help connect sellers to marketplaces. It has about 100,000 orders, with many sellers, customers, and other datapoints.
