# olist_ecommerce_sql
I use SQL with DB Browser for SQLite to analyze a dataset from Olist on Brazilian ecommerce.

This dataset is from the company Olist directly and on Kaggle. There are many different queries that one could do with the dataset, but I focused on queries that would be valuable for the company Olist. Olist is a company that provides ecommerce assistance to ecommerce sellers. They offer logistics and platform help. They are not a marketplace themselves. They help connect sellers to marketplaces.
